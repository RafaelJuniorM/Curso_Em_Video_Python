# Analise de String 

- len('string') => comprimento da frase 
- frase.count('a') => conta quantas vezes aparece a letra 'a' na frase
- frase.find('deo') => mostra em qual posição se incia a string 'deo'

# Transformação de String 

- frase.replace('Python', 'android') => substitui a palavra  'Python' por 'android'
- frase.upper() => transforma a letra em maiuscula 
- frase.lower() => transforma a letra em minuscula
- frase.capitalize() => somente a primeira letra maiuscula 
- frase.title() => transforma a primeira leta de cada palavra em maiuscula 
- frase.strip() => Elimina espaços vazios do inicio e final da string
- frase.rstrio() => Elimina espaços vazios da direita da string
- frase.lstrio() => Elimina espaços vazios da esquerda da string

# Divisão de String 

- frase.split() => ocorre divisão onde ocorre espaço vazios na string, transformando em lista.

# Junção de String

'-'.join(string) => adicionara o caractere '-' onde se encontra espaços vazios.
